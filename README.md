📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript that helps users manage daily tasks efficiently.

🎥 Project Reference Video:
To-Do List Project Tutorial :https://youtu.be/gRkaen6MeQc?si=Vn1pmfTN_0eiOl0f

📌 Features
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete tasks
🧠 Simple and user-friendly UI
⚡ Real-time updates without page reload
💾 (Optional) Local storage support for saving tasks
🛠️ Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation)

The project mainly uses JavaScript DOM manipulation to dynamically add, update, and remove tasks from the list.

📂 Project Structure
To-Do-List/
│── index.html
│── style.css
│── script.js
│── README.md
⚙️ How It Works
User enters a task in the input field
Clicks the Add button
Task is added to the list dynamically
User can:
Mark it as completed ✔️
Delete it ❌

This is achieved using:

Event listeners
DOM element creation
JavaScript functions
